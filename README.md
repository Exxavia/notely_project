# Notely Project

A Django-based collaborative note and task management system.

## Features

- User registration
- Email verification
- Login and logout
- Password reset
- User dashboard
- Profile management
- Avatar upload

- UI cleaned up
- Unit testing

## Installation

Clone the repository:

git clone <repo_url>

Create virtual environment:

python -m venv venv

Activate environment:

venv\Scripts\activate

Install dependencies:

pip install -r requirements.txt

Run migrations:

python manage.py migrate

Run server:

python manage.py runserver
